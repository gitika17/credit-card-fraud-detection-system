💳 Credit Card Fraud Detection System

Overview:-

This project uses machine learning to detect fraudulent credit card transactions based on transaction patterns and data analysis.

Features:-

* Data preprocessing and cleaning
* Handling imbalanced dataset
* ML models: Logistic Regression, Random Forest
* Performance evaluation using precision, recall, F1-score

Dataset:-

Dataset used: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Implementation:-

All implementation is available in `Fraud_Detection_Project.ipynb`

Tech Stack:-

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn
